# Borradores legales

> **BORRADORES — NO PUBLICAR SIN REVISIÓN PROFESIONAL.**

Estas plantillas no constituyen asesoría legal. Deben completarse con datos reales, prácticas efectivas y revisión jurídica aplicable en Colombia antes de publicarse.

- [Términos y condiciones](./TERMS_DRAFT.md)
- [Política de privacidad](./PRIVACY_DRAFT.md)
- [Tratamiento de datos personales](./DATA_PROCESSING_DRAFT.md)
- [Política de envíos](./SHIPPING_DRAFT.md)
- [Política de devoluciones](./RETURNS_DRAFT.md)
- [Política de cookies](./COOKIES_DRAFT.md)

Campos comunes por completar:

- Razón social o nombre del responsable.
- Identificación tributaria.
- Dirección, ciudad, correo y teléfono.
- Dominio definitivo.
- Canales para solicitudes y reclamos.
- Cobertura, horarios, tarifas y tiempos.
- Versión, fecha de vigencia y registro de cambios.
